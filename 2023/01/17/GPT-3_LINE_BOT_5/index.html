【GPT-3 LINE BOT】(5) 同場加映：介紹 Hugging Face 及 Meta AI BlenderBot

Author: tsamcservice Date: 一月 17, 2023  00:00:00

前言

當我在做 OpenAI API 的 GPT-3 LINE BOT 時後，我有發現了這 Hugging Face 有趣的網站，於是也稍微小玩一下，就順便分享出來，而這是我使用的是 facebook/blenderbot-3B 的模型。

  • 事先聲明：本人對於 Hugging Face 也是近期才發現，所以也算是挺陌生的

Hugging Face

官方網站
Hugging Face

簡單來說就是一個可以讓大家存放訓練資料集、訓練模型、開發工具的免費網站，包括自然語言處理、語音辨識及影像辨識等多種 AI 應用，真的可以說是機器學習界的 github ，直接減少進入機器學習的門檻

這位大神的一系列把 Hugging Face 講解得很清楚，也都是很棒的資源，有興趣的可以參考喔~
變形金剛與抱臉怪—NLP 應用開發之實戰 系列
另外對岸這網頁我覺得也說得很清楚
Huggingface 超詳細介紹

Meta AI

我相信大家對於 Meta 這公司一定不陌生，就是那個將川普帳號停權兩年的公司(最近好像要解封)，而 Meta 的舊名也是大家耳熟能詳，由馬克·祖克柏在大學與同學創立的 Facebook，Meta 公司目前旗下擁有Facebook、Instagram、WhatsApp 等社交軟體
至於 Meta AI 就是他們底下的一個專門處理演算法、自然語言處理、人工智慧系統研究、對話機器人等的人工智慧實驗室，其中也如同 OpenAI 公司一樣會定期的公布自己所開發的 AI 技術，而我們這次使用的 facebook/blenderbot 系列就屬於他們公司產出的

BlenderBot 3

另外，在 2022 年底(ChatGPT 在台灣火紅之前)Meta AI就有發布一個線上的對話機器人，不只沒多久 OpenAI 就公開 ChatGPT，又加上 BlenderBot 3 不開放給其他地區使用，所以身處在台灣的我們才會完全不知道

BlenderBot 3 對話機器人網站
https://geo-not-available.blenderbot.ai/
Hugging Face 中的 Blenderbot 模型
https://huggingface.co/docs/transformers/model_doc/blenderbot
